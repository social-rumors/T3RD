# T3RD-WWW2024

Code and datasets for our WWW 2024 paper - T3RD: Test-Time Training for Rumor Detection on Social Media.

## Requirements
```
python==3.6  
numpy==1.18.1  
torch==1.10.1
torch_scatter==2.0.9  
torch_sparse==0.6.12  
torch_cluster==1.5.9  
torch_geometric==2.0.3  
tqdm==4.40.0  
joblib==1.1.1
transformers==4.5.0
```

## Get Started
Run script
```
$ sh start.sh
```
